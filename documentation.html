Getting Started

  1. Create your account and complete your creator profile.
  2. Define one clear problem your agent or skill solves.
  3. Add usage examples and expected output format.
  4. Submit for review and iterate using moderation feedback.

What You Can Publish

  • Task-specific AI agents for support, writing, or automation.
  • Composable skills that can be chained in workflows.
  • Private drafts for team testing before public release.

Publishing Workflow

Submission Checklist

  • Clear title and short value statement.
  • Prompt safety boundaries and prohibited outputs documented.
  • At least 3 realistic examples.
  • Performance notes: expected latency and cost profile.

Trust and Quality Standards

  • No impersonation, fraud support, or harmful automation.
  • Disclose any external data sources used by the model.
  • Keep version notes transparent when behavior changes.
  • Repeated trust violations can suspend listing visibility.
